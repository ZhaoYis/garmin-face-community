## 新增需求

### 需求:Header 必须显示用户登录状态
页面顶部必须显示全局 Header 组件，包含网站名称、导航链接和用户状态。

#### 场景:未登录用户看到登录按钮
- **当** 用户未登录
- **那么** Header 右侧显示"登录"按钮，点击跳转到登录页面

#### 场景:已登录用户看到用户信息
- **当** 用户已登录
- **那么** Header 右侧显示用户头像和用户名

### 需求:已登录用户必须能看到用户下拉菜单
点击用户信息后必须显示下拉菜单，包含用户基本操作选项。

#### 场景:显示用户下拉菜单
- **当** 用户点击头像或用户名
- **那么** 显示下拉菜单，包含"个人中心"和"退出登录"选项

#### 场景:点击个人中心跳转
- **当** 用户点击下拉菜单中的"个人中心"
- **那么** 页面跳转到 /profile

#### 场景:点击退出登录
- **当** 用户点击下拉菜单中的"退出登录"
- **那么** 用户退出登录，页面刷新或跳转到首页

### 需求:Header 必须包含导航链接
Header 必须包含主要页面的导航链接。

#### 场景:导航链接可用
- **当** 用户查看 Header
- **那么** 可以看到"运动记录"、"创建海报"、"个人中心"等导航链接
