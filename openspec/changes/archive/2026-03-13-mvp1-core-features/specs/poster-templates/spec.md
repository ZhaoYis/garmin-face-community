## 新增需求

### 需求:系统必须提供预设海报模板
系统必须提供至少 4 套核心海报模板供用户选择。

#### 场景:模板列表展示
- **当** 用户进入海报创建页面
- **那么** 系统展示 4 套模板：成就海报、极简海报、艺术海报、越野海报

#### 场景:模板预览
- **当** 用户点击某个模板
- **那么** 系统展示该模板的预览图和适用场景说明

### 需求:每套模板必须有明确的设计规范
模板必须定义尺寸、数据字段、颜色方案等配置。

#### 场景:成就海报配置
- **当** 用户选择成就海报模板
- **那么** 系统应用深色+金色配色，显示比赛名称、完赛时间、平均配速、距离

#### 场景:极简海报配置
- **当** 用户选择极简海报模板
- **那么** 系统应用白底+黑字配色，仅显示核心数据

#### 场景:艺术海报配置
- **当** 用户选择艺术海报模板
- **那么** 系统应用渐变背景，适合社交媒体分享

#### 场景:越野海报配置
- **当** 用户选择越野海报模板
- **那么** 系统应用大地色系，突出累计爬升数据

### 需求:模板必须支持自定义字段
用户必须能够自定义海报中的个性标语。

#### 场景:自定义标语
- **当** 用户在海报编辑器中输入自定义标语
- **那么** 系统将标语渲染到海报指定位置
